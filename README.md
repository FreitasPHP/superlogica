# superlogica
Teste para o Processo Seletivo SuperLogica - PJBANK


1) Tecnologias e Requisitos
1.1) Banco de dados MYSQL.
1.2) PHP 7.2 ou superior.
1.3) Apache Server. 

2)Como rodar o Sistema
2.1) Clonar o repositório em https://github.com/FreitasPHP/superlogica.git
2.2) Executar o arquivo sql create_db_exercicio1, disponível na pasta ?app/_sql? em qualquer gerenciador de banco de dados mysql, exemplo: Workbench, Heidi, etc.
2.3) Editar o arquivo ?app/Config.php?, passando os parâmetros de login e senha do banco de dados nas constantes DB_HOST, DB_USER e DB_PASS.
2.4) Executar pelo navegador através do endereço do SERVER/superlogica(raiz), onde será chamado pelo browser o arquivo index.php.
