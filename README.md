# superlogica
Teste para o Processo Seletivo SuperLogica - PJBANK


? Tecnologias e Requisitos
1) Banco de dados MYSQL.
2) PHP 7.2 ou superior.
3) Apache Server. 

? Como rodar o Sistema
1) Clonar o repositório em https://github.com/FreitasPHP/superlogica.git
2) Executar o arquivo sql create_db_exercicio1, disponível na pasta ?app/_sql? em qualquer gerenciador de banco de dados mysql, exemplo: Workbench, Heidi, etc.
3) Editar o arquivo ?app/Config.php?, passando os parâmetros de login e senha do banco de dados nas constantes DB_HOST, DB_USER e DB_PASS.
4) Executar pelo navegador através do endereço do SERVER/superlogica(raiz), onde será chamado pelo browser o arquivo index.php.
